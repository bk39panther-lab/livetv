# Panther TV

Panther TV is a modern streaming platform designed to provide access to authorized live television channels, FAST channels, local programming, and user-friendly viewing experiences across web, mobile, and TV devices.

## Features

### Live TV

* Browse available channels
* Watch live streams
* Channel categories
* Featured channels

### Search

* Search channels by name
* Search by category
* Search by country

### Favorites

* Save favorite channels
* Sync favorites across devices

### Watch History

* Recently watched channels
* Continue watching

### TV Guide

* Electronic Program Guide (EPG)
* Current and upcoming programming

### User Accounts

* Google Login
* Apple Login
* Email Login

### Supported Devices

* Web Browser
* Android TV
* Fire TV
* Apple TV
* Android Mobile
* iPhone

## Technology Stack
